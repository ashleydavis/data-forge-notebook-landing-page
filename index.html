About Data-Forge Notebook

Data-Forge Notebook is a desktop application for Windows, Mac and Linux.

Use it to prototype JavaScript code in a highly visual fashion. It makes data transformation, analysis and visualization a breeze.

Features

  • Bring your own version of Node.js (or use the bundled version). Switch between Node.js using Node Version Manager (NVM).
  • Efficient lazy evaluation of code cells and caching of results.
  • Data formats: JSON, CSV, XML, YAML
  • Data formats: JSON, CSV, XML, YAML
  • Databases: Microsoft SQL, MySQL, PostgressSQL, MongoDB and more being added all the time
  • Live visualization for tables, json, html and charts
  • JavaScript visualization libraries: c3, Canvas.js, Highcharts and more
  • Easily do web scraping
  • Install and use arbitrary Node.js modules
  • Create projects with your own ad-hoc JavaScript module to compliment and extend your Notebook code
  • Export to the web. Host your visualization on any server, or easily include it in your blog
  • Export to a Node.js project and deploy to your production JavaScript environment
  • Export screenshots of your charts - great for including charts in your blog
  • Export your notebooks to PDF files - great for printing student handouts
  • Extensive library of examples, data and snippets to help get you started.
  • Contribute your own example code and help others to get started.
  • Works with your favorite JavaScript version of derivative: ES6, TypeScript, Flow, CoffeeScript, etc. New languages are easily supported.
  • Manage your whole exploratory coding project - notebooks, script files, dependencies and data.
  • Integrates with Git and Mercurial for version control.

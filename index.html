About Data-Forge Notebook

Data-Forge Notebook is a desktop application for Windows, Mac and Linux. You use it to prototype JavaScript code in a highly visual fashion. It is easy to transform, manipulate and visualize any kind of structured data.

What else do you get?

  • It works with various data formats (JSON, CSV, XML, YAML), databases (SQL, MongoDB) and common JavaScript visualization libraries. Is it missing your favourite data format or database? Let me know and I'll add it.
  • Install and include arbitrary Node.js module and also ad-hoc JavaScript modules to complement your own code.
  • Your code can be exported to the web (host on any server, or easily include it in your blog?), or exported to a Node.js project. You can easily reuse your code in your production JavaScript environment.
  • Export screenshots of your charts and data to image files (great for charing images on your blog).
  • Export your complete notebook to PDF files (great for printing student handouts).
  • Browse an extensive library of example code, data and snippets to help get you started. Contribute your own example code and help others to get started.
  • Oh also, Data-Forge Notebook also works with your favorite JavaScript version of derivative: ES6, TypeScript, Flow, CoffeeScript, etc. Your favourite language missing? Let me know and I'll add it.
